## 这个人

在下面你可能会找到一个机械化的版本的我：

```python
class thisHuman():
    def __init__(self):
        self.name = 'Yunan Wang'
        self.location = 'Paris, France'
        self.occupation = 'master student'
        self.field = 'Sustainability'
        self.languages = {
            '普通话': 'native',
            'English': 'professional',
            'Français': 'professional',
            'Italiano': 'beginner',
            'Chinese dialects': ('Cantonese', 'Teochew')
        }
```

然而，我更喜欢一个更“有机”的解释，即使用一组标签或形容词简化地概括一个人是过于简单的。

你可能首先在首页上看到的自我介绍其实是我更倾向于的一种概括方式：

> 一个**_极简主义者_**，
>
> 一个**_可持续发展_**的倡导者，
>
> 以及一个业余的**_程序员_**

我认为这三个标签在很有限的程度上概括了我的个性，兴趣领域和人生目标。然而，请随意阅读我的文章，以对这个人有更微妙的理解。

- 摆弄我的云服务器（这个网站就是这个努力的成果）

## 这个网站

这个网站既用于介绍我自己，也用于分享我的想法和经验。我预计访客可能有文化和语言多样的背景，所以我不会限制用某种语言写作；但信息可能最容易用英语获取。（我希望能早点写一篇文章。）

网站不会有任何主要主题，但我预计它们将如下：

- 对生活和其他无意义事物的反思
- 技术分享
- 对某些事件的看法

截至 2023 年 10 月，该网站是用[Astro](https://astro.build/)从零开始构建的，并托管在 Cloudflare Pages 上。
