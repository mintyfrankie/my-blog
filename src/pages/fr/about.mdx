# A Propos

Salut, étranger. Bienvenue sincère dans cet abri sur Internet.

## L'être humain

Ci-dessous, vous trouverez une version mécanique de moi :

```python
class thisHuman():
    def __init__(self):
        self.name = 'Yunan Wang'
        self.location = 'Paris, France'
        self.occupation = 'master student'
        self.field = 'Sustainability'
        self.languages = {
            '普通话': 'native',
            'English': 'professional',
            'Français': 'professional',
            'Italiano': 'beginner',
            'Chinese dialects': ('Cantonese', 'Teochew')
        }
```

Cependant, je préfère une interprétation plus "organique" de moi, même si c'est simpliste de résumer un individu par un ensemble de tags ou d'adjectifs.

L'auto-introduction que vous pourriez d'abord voir sur la page d'accueil est en quelque sorte une manière de généralisation que je préfère :

> un **_Minimaliste_**,
>
> un défenseur de la **_Durabilité_**,
>
> et un **_programmeur_** amateur

Je pense que ces trois tags résument, dans une mesure très limitée, ma personnalité, mes domaines d'intérêt et mes objectifs de vie. Néanmoins, n'hésitez pas à lire mes articles pour avoir une compréhension subtile de cet humain.

**Ce que je fais actuellement**

- Apprendre l'italien (je viens de terminer un cours pour débutants)
- Bidouiller mon serveur cloud (ce site est le fruit de cet effort)
- M'entraîner pour être capable de m'inscrire à un semi-marathon l'année prochaine

## Le site

Ce site sert à la fois à me présenter et à partager mes pensées et expériences. Je m'attends à ce que les visiteurs aient un arrière-plan culturel et linguistique diversifié, donc je ne me limiterai pas à écrire dans une certaine langue ; mais les informations seront probablement plus accessibles en anglais. (J'espère pouvoir écrire un article tôt ou tard.)

Il n'y aura pas de thèmes majeurs pour le site, mais je m'attends à ce qu'ils soient les suivants :

- Réflexions sur la vie et autres futilités
- Partage technique
- Pensées sur certains événements

En date d'octobre 2023, le site est construit de zéro avec [Astro](https://astro.build/) et hébergé sur Cloudflare Pages.
